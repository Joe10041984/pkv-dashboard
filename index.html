Geräte-Sync über GitHub
Personal Access Token mit „gist“-Berechtigung eintragen. Gist-ID leer lassen — „Jetzt hochladen“ legt automatisch einen privaten Gist an. Das Token bleibt nur auf diesem Gerät; danach gleicht die App beim Öffnen und nach jeder Änderung automatisch ab. Fürs Beleg-Repo braucht das Token zusätzlich „Contents: Read/Write“ auf genau diesem Repo.
